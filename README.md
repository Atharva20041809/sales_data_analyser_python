# Sales Data Analysis using Python

This project analyzes sales data stored in a CSV file using core Python. It processes raw data and generates useful insights such as revenue analysis and sales trends without using any external libraries.

## Features
- Read and process CSV sales data
- Calculate revenue per product
- Identify the highest revenue day
- Find the most sold product
- Compute average order value
- Modular code structure using utility functions

## Project Structure

sales-data-analysis-python/
├── main.py
├── utils.py
├── sales.csv
└── README.md

## Dataset
The dataset (sales.csv) contains the following fields:
- Product
- Quantity
- Price
- Date
- Total revenue

## How to Run
Make sure Python 3 is installed, then run:
python main.py
